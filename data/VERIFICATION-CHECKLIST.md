## 📞 故障排除

### 问题：PDF 无法下载
**诊断步骤：**
1. 检查 `Active/` 文件夹是否存在对应的 PDF 文件
2. 检查文件名是否与 JSON 中的名称完全匹配（包括大小写）
3. 检查文件是否有读取权限

### 问题：页面显示为空
**诊断步骤：**
1. 打开浏览器开发者工具 (F12)
2. 查看 Console 选项卡中是否有错误信息
3. 检查 Network 选项卡中文件是否正确加载

### 问题：语言显示不正确
**诊断步骤：**
1. 检查浏览器是否支持 JavaScript
2. 检查 i18next 库是否正确加载
3. 清除浏览器缓存并重新加载页面
